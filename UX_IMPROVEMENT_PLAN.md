**位置**：占据剩余空间
**内容**：

**布局**：
- 响应式网格 (1-4 列)
- 自适应卡片大小
- 虚拟滚动优化

### 4. 底部状态栏 (32px)
**位置**：固定在底部

**信息**：

- 当前操作

## 🔧 核心功能

### 1. 保存功能
**触发方式**：
- 工具栏保存按钮
- 快捷键 Cmd/Ctrl + S
- 自动保存（每 30 秒）

**保存内容**：
```json
{
  "name": "项目名称",
  "characters": [...],
  "storyContent": "...",
  "storyOutline": "...",
  "storyboards": [...],
  "createdAt": "2024-01-01T00:00:00Z",
  "updatedAt": "2024-01-01T00:00:00Z"
}
```

**存储位置**：
- localStorage (自动保存)
- JSON 文件 (手动导出)

### 2. 重试功能
**适用场景**：
- 单个分镜生成失败
- 网络错误
- API 超时

**操作流程**：
1. 点击失败卡片的"重试"按钮
2. 状态变为"生成中"
3. 重新调用生成 API
4. 更新状态和图片

**UI 反馈**：
- 失败状态：红色边框 + 错误图标
- 重试按钮：蓝色高亮
- 加载动画：旋转图标

### 3. 预览功能
**触发方式**：
- 点击卡片的"预览"按钮
- 点击图片区域

**预览方式**：
- 新标签页打开原图
- 或使用模态框展示
- 支持缩放和下载

### 4. 新建项目
**操作流程**：
1. 点击"新建项目"按钮
2. 确认对话框（如有未保存数据）
3. 清空所有状态
4. 重置到步骤 1

**数据处理**：
- 清空 localStorage
- 重置所有 state
- 刷新页面

## 📊 状态管理

### 项目状态
```typescript
interface ProjectState {
  name: string;
  currentStep: number;
  characters: Character[];
  storyContent: string;
  storyOutline: string;
  storyboards: Storyboard[];
  isAnalyzing: boolean;
  isGenerating: boolean;
}
```

### 分镜状态
```typescript
type StoryboardStatus =
  | 'pending'    // 等待中
  | 'generating' // 生成中
  | 'completed'  // 已完成
  | 'failed';    // 失败
```
